# Визуализатор графа зависимостей (Вариант №8)

## Общее описание
Инструмент для визуализации графа зависимостей пакетов Ubuntu (apt). На этапе 1 - минимальный CLI-прототип с конфигурацией из TOML.

## Описание функций и настроек
- Чтение конфига из TOML (параметры: package, url_or_path, test_mode, version, filter_substring).
- Обработка ошибок: Проверка наличия, типов, пустых значений.
- Вывод параметров (только этап 1).

## Команды для сборки и запуска
- Сборка: javac src/Main.java
- Запуск: java -cp src Main
- Тест ошибок: Измените config.toml (удалите ключ, задайте неверный тип) и запустите.

## Примеры использования
1. С валидным config.toml: Выводит параметры.
2. С ошибкой (отсутствует ключ): Выдаёт сообщение об ошибке.
3. С неверным test_mode: "test_mode = maybe" - ошибка типа.
